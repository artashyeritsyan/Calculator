# Calculator
---
Write a calculator that receives a string, for example “2 + 2”, and at the output gives the answer to the expression, in this case the answer will be “4”.
```
Example 1`
input: "16 / 4"
output: 4

Example 2`
input: "6*2"
output: 12
```
Bonus
```
Add logic for the following operators` ^, %.
```
